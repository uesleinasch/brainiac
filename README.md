Este é um projeto para a criação de meu segundo cérebro usando arquivo .md e
Claude.

As informações aqui deve ser salvas em formato tokenizado para otimizar buscas e
interações futuras.

O sistema de arquivos é baseado nas pastas de:

- **longMemory**: Retenção da informação durante longos períodos de tempo que podem prolongar-se toda a vida.
- **shortMemory**: Armazenamento da informação por um curto espaço de tempo (até ≈ 20 segundos) graças à repetição ativa.
  Inclui a memória de trabalho, um sistema que mantém e opera mentalmente a informação no momento presente, organizando-a de forma contínua e recuperando-a quando convém.
- **semanticMemory**: Envolve os conhecimentos organizadores do mundo, por exemplo, lembramo-nos do dia que o Brasil foi descoberto e quem o descobriu.
- **memoryTransfer**: Sistema que organiza e transita informações entre as memorias
